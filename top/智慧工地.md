# 智慧工地对接接口

1. 获取项目纪要列表->**GET**  http://模块/project/checkList
   ```
   参数:startTime endTime string 非必填
   checkName string🚀️纪检名称
   checkTypeId:string 纪检类型 
   checkTypeName:string 纪检类型名称 采购类别,合同列别
   checkWay:string 3种纪检方式->保存1,2,3 
       1.代表事前(供方选择)
       2.代表事中(合同,合同类别)
       3.代表事后(合同,超结算(和朱确定))
   json: {
   	"code": 0,
   	"data": [
   		{
   			"id": "",
   			"checkName": "",
   			"checkTypeId": "",
   			"checkTypeName": "",
   			"checkWay": "",
   			"checkDate": "",
   			"checkSquare": "",
   			"projectId": "",
   			"projectName": "",
   			"tenantKey": "",
   			"tenantName": "",
   			"ancestors": "",
   			"createdTime": "",
   			"updatedTime": ""
   		}
   	],
   	"msg": ""
   }
   ```
